CORE PINBALL VILLAINS (1967 SERIES ONLY)

These are your main mode anchors — the ones that can carry full gameplay modes.

🧠 Doctor Octopus

Best use: control / capture mode

Multiple appearances in Season 1 + Season 3
Mechanical arms = perfect for shot locking or staged multiball
Mode idea: “Tentacle Lockdown”
Certain shots get “grabbed” (locked out)
Player must free lanes to continue scoring



🧪 Green Goblin

Best use: chaos / risk mode

Psychological + gadget-based attacks
Illusions, unpredictability, traps
Mode idea: “Goblin Chaos”
Random shots light/unlight
Mystery awards / fake jackpots


⚡ Electro

Best use: speed / combo mode

Travels through electricity, fast and aggressive
Repeated appearances
Mode idea: “Power Surge”
Combo shots build charge meter
Timed jackpot when fully charged


🦅 Vulture

Best use: orbit / aerial flow mode

Air-based attacks, bird coordination
Strong visual identity (flight loops)
Mode idea: “Sky Attack”
Orbit shots = points multiplier
Keep ball moving = survival bonus


🪨 Rhino

Best use: impact / bash mode

Brute force, straightforward destruction
Appears multiple times
Mode idea: “Charge Impact”
One-shot jackpots (ramps / bash targets)
Build-up → massive payoff


🧲 Scorpion

Best use: precision danger mode

High aggression, unstable strength
Good “punish sloppy play” villain

Tight shot windows
Miss = timer penalty


🏖️ Sandman

Shape-shifting, impersonation

Random shot behavior changes

🎭 Mysterio

👑 Kingpin

Organized crime mastermind

🧠 CORE DESIGN RULE (IMPORTANT)
Don’t think “mode per villain.”
Think:
“What does the player DO differently for this villain on THIS layout?”

If you don’t change shot behavior, the mode feels identical.

🕷️ MODE MAPPING TO YOUR LAYOUT
🧠 Doc Ock — Control / Lockdown Mode

Doctor Octopus

Use: center + main lanes

“Arms” = 2–3 shots temporarily disabled or reversed scoring
Player must hit alternating left/right banks to “break grip”
Final shot = center “arm core” target
Reward: opens all shots for combo burst

👉 This becomes your mechanical restriction mode

🧪 Green Goblin — Chaos Mode

Green Goblin

Use: ALL shots, randomized logic

Shots randomly light/unlight every few seconds
“Pumpkin bomb” inserts = hurry-up jackpots
Some shots become fake (no score or negative multiplier unless corrected)

👉 On this layout: chaos works because geometry is predictable

⚡ Electro — Combo / Flow Mode

Electro

Use: orbit/loop + return lanes

Left orbit → right return = combo chain
Each successful loop increases “charge meter”
Full charge unlocks center jackpot

👉 This is your flow state mode

🦅 Vulture — Orbit Mastery Mode

Vulture

Use: outer lanes / orbits

Only orbit shots score
Inner targets temporarily “unsafe” (penalty or low value)
Keeping ball in upper playfield = multiplier growth

👉 Forces clean, controlled shooting

🪨 Rhino — Bash Mode

Rhino

Use: standup banks / central targets

Only direct hits count (no combos needed)
Each bank hit = “impact damage”
3–5 hits = “charge breakthrough” multiball

👉 Perfect for Gottlieb-style satisfying feedback

🧲 Scorpion — Precision Punish Mode

Scorpion

Use: narrow shot lanes

Only 1–2 safe shots lit at a time
Wrong shot = timer penalty or reset progress
Success builds “venom meter” to jackpot

👉 This makes players care about accuracy

🏖️ Sandman — Physics Distortion Mode

Sandman

Use: all shots, but unstable rules

Certain shots randomly “sink” (temporary dead scoring)
Others become high-value unexpectedly
Ball save or kickback triggers unpredictably

👉 On this layout: this becomes controlled randomness

🎭 Mysterio — Illusion Mode

Mysterio

Use: shot deception layer

Fake-lit shots appear (LED tricking player)
Only 1 real shot is correct at any time
Wrong hits reset combo or spawn new illusion

👉 This is your mind game mode

👑 Kingpin — Final Boss Mode

Kingpin

Use: full playfield progression

Phase 1: hit all banks (crime network)
Phase 2: lock 3 shots (money laundering system)
Phase 3: center “Kingpin shot” opens
Phase 4: all shots worth jackpot for limited time

👉 This is your wizard-style endgame

Lizard

Parafino

Fakir

⚠️ REAL DESIGN ADVICE (IMPORTANT)

The Gottlieb layout is:

Not wide
Not modern flow-heavy
Very stop-and-shoot friendly

So:

DO:
Use standups as “villain identity anchors”
Make modes change RULES, not geometry
Keep 1–2 shot priorities per mode
DON’T:
Try to simulate modern deep ramp stacking
Overuse multi-layer objectives
Add too many simultaneous lit states
