# 🧲 Scorpion — FINAL STRUCTURE (WITH REAL RISK CHOICE)

## Core loop (unchanged and correct)

* spinner builds **Venom Meter**
* Venom full → “Sting Ready” state
* player chooses LEFT or RIGHT exit
* each exit leads to a single drop target collect


# 🎯 EXIT MEANING (THIS IS THE KEY DESIGN WIN)

## LEFT EXIT (HARD MODE)

* leads to **RIGHT drop target**
* harder shot geometry / worse feed / tighter timing
* higher reward

### Behavior:

* must hit right drop target within short window
* success = BIG Sting Jackpot (Venom x higher multiplier)

---

## RIGHT EXIT (EASY MODE)

* leads to **LEFT drop target**
* easier shot / safer return / cleaner alignment
* lower reward

### Behavior:

* same mechanic, more forgiving execution
* lower Sting Jackpot multiplier
