# 🪨 RHINO MODE
Mode: Rhino Bash

** build Rhino's rage (pops), build JP value by smashing targets, collect JPs
** builld anger as high as possible, then cash before he loses control
** collect JP 3 times

## 1. RHINO RAGE METER
| Rage Stage  | Pops Required to Reach |
| ----------- | ---------------------- |
| 1 Calm      | start mode             |
| 2 Irritated | 2 pops                 |
| 3 Charging  | 4 pops                 |
| 4 Enraged   | 7 pops                 |
| 5 Berserk   | 10 pops                |

Each stage increases the amount the target switches will add to the jackpot.

Example:

| Rage Stage | Add Per Target |
| ---------- | -------------- |
| 1          | 2K             |
| 2          | 5K            |
| 3          | 15K            |
| 4          | 50K            |
| 5          | 100K           |

At Stage 4:

* lights begin pulsing yellow/red
* Rhino voice/callout: “RRRHHH!”



## 2. RHINO JACKPOT VALUE 

Starts at:
* 50K 

### Rhino Smash switches adds current RHINO RAGE METER amount to JP

* both drop banks
* both web standups
* saucers
* spinner
* upper rollover button
* A/B top lanes

- can collect at any time



## 3. MAX RAGE

At Rage Stage 5 the berserk timer starts (8 sec).

During those 8 sec:

* every target hit adds 100K
* lamps pulse aggressively
* timer on screen



## 4. RHINO CRASH (RHINO JACKPOT MISSED)

If player does NOT collect Left B before 8 sec timer expires:

* jackpot resets to base (50K)
* rage resets to stage 2
* light show - blackout? / crash sound



## 5. RHINO JACKPOT COLLECT

- successful LEFT MIDDLE B LANE collect Jackpot

* Jackpot resets
* JP1 base = 50K
* JP2 base = 100K
* JP3 base = 150K

* does NOT reset rage fully. 5->4, 4->3, 3->2, 2->2, 1->1





Perfect.

So Rhino Jackpot Collect = `s_inlane_m_l`.

That gives us everything needed.

I’m going to build this in **MPF 0.80 / config_version=6 style**, with:

* mode start/stop
* player vars
* rage thresholds
* berserk timer
* jackpot build
* jackpot collect
* crash handling
* 3 jackpot victory

This will be a genuine working skeleton, not pseudo-code.

## `modes/rhino/config/rhino.yaml`

```yaml
#config_version=6

mode:
  start_events: start_mode_rhino
  stop_events: rhino_mode_complete, rhino_mode_failed
  priority: 400


player_vars:
  rhino_pop_count:
    value_type: int
    initial_value: 0
  rhino_rage_stage:
    value_type: int
    initial_value: 1
  rhino_add_value:
    value_type: int
    initial_value: 2000
  rhino_jackpot:
    value_type: int
    initial_value: 50000
  rhino_jackpots_collected:
    value_type: int
    initial_value: 0


timers:
  rhino_berserk_timer:
    start_value: 8
    end_value: 0
    direction: down
    tick_interval: 1s
    control_events:
      - action: start
        event: rhino_berserk_start
      - action: stop
        event: rhino_jackpot_collect
      - action: stop
        event: rhino_berserk_crash


# -------------------------
# POPS BUILD RHINO RAGE
# -------------------------

variable_player:
  s_pop_left_active:
    rhino_pop_count: 1

  s_pop_right_active:
    rhino_pop_count: 1


event_player:

  s_pop_left_active{current_player.rhino_pop_count>=2 and current_player.rhino_rage_stage==1}: rhino_rage_2
  s_pop_right_active{current_player.rhino_pop_count>=2 and current_player.rhino_rage_stage==1}: rhino_rage_2

  s_pop_left_active{current_player.rhino_pop_count>=4 and current_player.rhino_rage_stage==2}: rhino_rage_3
  s_pop_right_active{current_player.rhino_pop_count>=4 and current_player.rhino_rage_stage==2}: rhino_rage_3

  s_pop_left_active{current_player.rhino_pop_count>=7 and current_player.rhino_rage_stage==3}: rhino_rage_4
  s_pop_right_active{current_player.rhino_pop_count>=7 and current_player.rhino_rage_stage==3}: rhino_rage_4

  s_pop_left_active{current_player.rhino_pop_count>=10 and current_player.rhino_rage_stage==4}: rhino_berserk_start
  s_pop_right_active{current_player.rhino_pop_count>=10 and current_player.rhino_rage_stage==4}: rhino_berserk_start


# -------------------------
# RAGE STAGE ADVANCEMENT
# -------------------------

variable_player:
  rhino_rage_2:
    rhino_rage_stage: 2
    rhino_add_value: 5000

  rhino_rage_3:
    rhino_rage_stage: 3
    rhino_add_value: 15000

  rhino_rage_4:
    rhino_rage_stage: 4
    rhino_add_value: 40000

  rhino_berserk_start:
    rhino_rage_stage: 5
    rhino_add_value: 100000


# -------------------------
# RHINO SMASH SHOTS BUILD JACKPOT
# -------------------------

variable_player:

  s_web_target_left_active:
    rhino_jackpot: current_player.rhino_add_value

  s_web_target_top_active:
    rhino_jackpot: current_player.rhino_add_value

  s_saucer_1_active:
    rhino_jackpot: current_player.rhino_add_value

  s_saucer_2_active:
    rhino_jackpot: current_player.rhino_add_value

  s_saucer_3_active:
    rhino_jackpot: current_player.rhino_add_value

  s_web_spinner_active:
    rhino_jackpot: current_player.rhino_add_value

  s_star_rollover_active:
    rhino_jackpot: current_player.rhino_add_value

  s_inlane_a_active:
    rhino_jackpot: current_player.rhino_add_value

  s_left_drops_1_active:
    rhino_jackpot: current_player.rhino_add_value
  s_left_drops_2_active:
    rhino_jackpot: current_player.rhino_add_value
  s_left_drops_3_active:
    rhino_jackpot: current_player.rhino_add_value

  s_right_drops_1_active:
    rhino_jackpot: current_player.rhino_add_value
  s_right_drops_2_active:
    rhino_jackpot: current_player.rhino_add_value
  s_right_drops_3_active:
    rhino_jackpot: current_player.rhino_add_value
  s_right_drops_4_active:
    rhino_jackpot: current_player.rhino_add_value
  s_right_drops_5_active:
    rhino_jackpot: current_player.rhino_add_value


# -------------------------
# JACKPOT COLLECT
# -------------------------

event_player:
  s_inlane_m_l_active: rhino_jackpot_collect


variable_player:
  rhino_jackpot_collect:
    score: current_player.rhino_jackpot
    rhino_jackpots_collected: 1


# increase next base jackpot after each collect
  rhino_jackpot_collect{current_player.rhino_jackpots_collected==1}:
    rhino_jackpot: 100000

  rhino_jackpot_collect{current_player.rhino_jackpots_collected==2}:
    rhino_jackpot: 150000

  rhino_jackpot_collect{current_player.rhino_jackpots_collected>=3}:
    rhino_jackpot: 200000


# rage drops one level after successful collect
  rhino_jackpot_collect{current_player.rhino_rage_stage==5}:
    rhino_rage_stage: 4
    rhino_add_value: 40000

  rhino_jackpot_collect{current_player.rhino_rage_stage==4}:
    rhino_rage_stage: 3
    rhino_add_value: 15000

  rhino_jackpot_collect{current_player.rhino_rage_stage==3}:
    rhino_rage_stage: 2
    rhino_add_value: 5000

  rhino_jackpot_collect{current_player.rhino_rage_stage==2}:
    rhino_rage_stage: 1
    rhino_add_value: 2000


event_player:
  rhino_jackpot_collect{current_player.rhino_jackpots_collected>=3}: rhino_mode_complete


# -------------------------
# BERSERK CRASH
# -------------------------

event_player:
  timer_rhino_berserk_timer_complete: rhino_berserk_crash


variable_player:
  rhino_berserk_crash:
    rhino_jackpot: 50000
    rhino_rage_stage: 2
    rhino_add_value: 5000
    rhino_pop_count: 2


# -------------------------
# OPTIONAL SHOW HOOKS
# -------------------------

show_player:
  start_mode_rhino:
    rhino_mode_running:
      loops: -1

  rhino_berserk_start:
    rhino_berserk_show:
      loops: -1

  rhino_berserk_crash:
    rhino_crash_show:
      loops: 0

  rhino_mode_complete:
    rhino_win_show:
      loops: 0
```
