# CLEAN RULE MODEL

You actually need **two separate values**:

---

## 1. RHINO JACKPOT VALUE

This is what Left B collects.

Starts at maybe:

* 25K or 50K

Every scoring target (except pops) adds:

* current rage-based add amount

---

## 2. RHINO RAGE METER

Built by pops only.

Each pop hit increases rage stage:

* Stage 1 = calm
* Stage 2 = irritated
* Stage 3 = charging
* Stage 4 = enraged
* Stage 5 = berserk (danger zone)

Each stage multiplies jackpot adds.

Example:

| Rage Stage | Add Per Target |
| ---------- | -------------- |
| 1          | 5K             |
| 2          | 10K            |
| 3          | 20K            |
| 4          | 35K            |
| 5          | 50K            |

Now pops are meaningful without directly being score generators.

# THE CRASH MECHANIC (ESSENTIAL)

Need a clear overbuild point.

### Suggested:

At Rage Stage 5:
a hidden Rhino instability timer starts (say 6 sec).

During those 6 sec:

* every target hit adds huge value
* lamps pulse aggressively

If player does NOT collect Left B before timer expires:

## RHINO CRASH

* jackpot resets to base
* rage resets to stage 1
* maybe brief lamp blackout / crash sound

This makes Berserk thrilling.
